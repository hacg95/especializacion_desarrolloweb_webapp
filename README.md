# Static Web App

This project is a simple static web application built using HTML, CSS, and JavaScript. It serves as a template for creating web applications with a clean structure and organized files.

## Project Structure

```
static-web-app
├── src
│   ├── index.html        # Main HTML document
│   ├── css
│   │   └── styles.css    # Styles for the web app
│   ├── js
│   │   └── main.js       # JavaScript code for interactivity
│   └── assets
│       ├── fonts         # Directory for font files
│       └── data          # Directory for data files
├── .gitignore            # Files to be ignored by version control
├── package.json          # npm configuration file
└── README.md             # Documentation for the project
```

## Getting Started

To get started with this project, follow these steps:

1. **Clone the repository**:
   ```
   git clone <repository-url>
   ```

2. **Navigate to the project directory**:
   ```
   cd static-web-app
   ```

3. **Install dependencies** (if any):
   ```
   npm install
   ```

4. **Open the `src/index.html` file in your browser** to view the application.

## Usage

- Modify the `src/css/styles.css` file to change the styles of the application.
- Update the `src/js/main.js` file to add interactivity and client-side logic.
- Place any font files in the `src/assets/fonts` directory.
- Store any necessary data files in the `src/assets/data` directory.
